# Contributing

Thanks for your interest. This is primarily an internal repo for Digital Hooligan LLC.

## Local Development
From repo root:

- Install deps: `pnpm install`
- Dev server: `pnpm -C apps/digitalhooligan-web dev`
- Build: `pnpm -C apps/digitalhooligan-web build`

## Pull Requests
- Keep PRs focused and small.
- Include screenshots for UI changes.
- Ensure CI passes (lint/typecheck/build).
